Release Notes for Cairo-client v0.1:
=============================

Cairo client provides an open source speech client library written entirely in the Java programming language.  
To achieve maximum compatibility with existing and future speech servers it has been designed from the ground 
up to comply with the MRCPv2 standard. (For more information on MRCPv2 please 
see http://www.ietf.org/internet-drafts/draft-ietf-speechsc-mrcpv2-14.txt)

This is the first release for the cairo-client project.  This release provides a 
libarary of basic speech capabilities.  It uses MRCPv2 to execute communicate with
a MRCPv2 server and execute the requests.  You can use cairo-client to build 
speech applications.
